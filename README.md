# rag-code-documentation-navigator
A Retrieval-Augmented Generation (RAG) system to navigate, search, and explain large code repositories using embedding and vector databases.
# Project Documentation

This document explains the architecture, data flow, and implementation
details of the Code Documentation Navigator (RAG) project.

## Architecture
- Code ingestion from GitHub repositories
- Chunking and embedding generation
- Vector storage using FAISS
- Retrieval + LLM-based answer generation

## Future Work
- VS Code extension
- Multi-repo support
- Agent-based reasoning
